# תשובה לשאלה: האם יש פה משהו חשוד? וירוס? משהו?

## תשובה קצרה: **לא, אין וירוס**

אחרי בדיקה מקיפה של הקוד, **לא נמצא וירוס או קוד זדוני**. האפליקציה GeminiDesk היא תוכנה לגיטימית.

---

## אבל יש כמה דברים שכדאי לדעת:

### 1. 📦 תוסף Chrome צד שלישי

האפליקציה כוללת תוסף Chrome בשם "MCP SuperAssistant" (בתיקייה `0.5.8_0/`) שמספק פונקציונליות MCP.

**מה זה עושה?**
- מאפשר שילוב עם Model Context Protocol
- נטען אוטומטית לאפליקציה
- אוסף נתוני שימוש דרך Google Analytics

**איך לכבות?**
1. הגדרות → מתקדם
2. בטל סימון "Load Unpacked Extension"
3. הפעל מחדש

### 2. 📊 איסוף נתונים אנליטיים

התוסף שולח מידע שימוש ל-Google Analytics:
- מזהה ייחודי של המשתמש
- מידע על דפדפן ומערכת הפעלה
- שימוש בכלי MCP
- דיווחי שגיאות

**אין איסוף של:**
- סיסמאות או מידע אישי
- תוכן השיחות שלך
- קבצים אישיים

### 3. 🔧 פורט Debug פתוח

האפליקציה פותחת פורט debug של Chrome (9222) שיכול לאפשר בדיקת האפליקציה מהרשת המקומית.

### 4. 🐛 פגיעויות בתלויות

**תוקנו:**
- ✅ glob - פגיעות ב-Command Injection
- ✅ js-yaml - Prototype Pollution
- ✅ tmp - בעיית Symlink

**נותרו:**
- ⚠️ Electron גרסה 28.3.3 (מומלץ לשדרג ל-35.7.5+)
